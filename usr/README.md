# Manajemen Mandiri Pasien ACS

Aplikasi ini adalah sistem manajemen kesehatan mandiri untuk pasien yang sedang dalam masa pemulihan setelah mengalami Sindrom Koroner Akut (Acute Coronary Syndrome - ACS). Aplikasi membantu memantau kesehatan harian, jadwal obat, serta menyediakan panduan rencana darurat yang cepat dan mudah diakses.

## Fitur Utama

*   **Dasbor Kesehatan:** Ringkasan informasi dan akses cepat ke fitur pemantauan kesehatan.
*   **Pengingat Obat:** Pantau kepatuhan meminum obat rutin harian seperti aspirin, clopidogrel, dan obat-obatan jantung lainnya.
*   **Pantauan Tanda Vital:** Catat tekanan darah, denyut nadi, dan berat badan untuk melihat tren kesehatan.
*   **Rencana Tindakan Darurat:** Panduan langkah demi langkah saat mengenali gejala serangan jantung berulang serta akses instan ke kontak layanan darurat.

## Tech Stack

*   **Framework:** Flutter (Android, iOS, Web)
*   **Bahasa:** Dart
*   **Desain:** Material 3 (Tema dominan Merah untuk kesadaran kardiovaskular)

## Cara Menjalankan

1.  Pastikan Anda telah menginstal Flutter SDK (versi 3.0 ke atas).
2.  Jalankan `flutter pub get` untuk mengambil dependensi.
3.  Jalankan `flutter run` untuk memulai aplikasi di perangkat yang terhubung atau emulator.
